===============
extract_data.py
===============
---------------------------------------------------------------
* Run method > python extract_data.py [ option 1 ] [ option 2 ]
---------------------------------------------------------------
* option 1 : dcm directory in 1-Dataset.  ex)dcm_500 
---------------------------------------------------------------
* option 2 : choice a desease.   ex)epidural
---------------------------------------------------------------
* 6 types of desease
 - epidural
 - intraparenchymal
 - intraventricular
 - subarachnoid
 - subdural
 - any
---------------------------------------------------------------
* Results are saved in ./[option 1 ]_res/
---------------------------------------------------------------
